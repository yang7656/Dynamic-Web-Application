ABOUT THE WEB APP

API USED FOR THIS WEBSITE

We use Leaflet API for creating the map and add all the feature to the map, like markers and panning options.
We also use Nominatim API for searching address that user can input through the search box or by clicking at a specific crime.
Our data come from a Restful Api that we built. The data come from the St. Paul Crime Dataset. We access the data through a simple query that permit the website to populate the map with the neighborhood marker and the crime table.
We based our website design on the CSS framework Skeleton, because it is easy to use and mobile friendly.

Some Interesting finding in our data
  1. No Homicide happened on October 2019
  2. Overall, the farther north it goes, the more incident type of Weapons, Discharging a Firearm in the City Limits
  3. Some address need to add a space so that it can be searched in nominatim. e.g 430 DOROTHY DAY PL, St.Paul, Minnesota
  4. From January 1st 2018 to January 1st 2019 the neighborhoods with the highest number of theft is Union Park with 341 crimes
  5. In the month of April 2018 there have been committed 310 Crimes in Union Park, While in April of 2019 there have been committed 700 crimes in Union Park
  6. In 2018 there have been 90 Robberies in St. Paul, Macalester-Groveland is the only one that registered 0.
